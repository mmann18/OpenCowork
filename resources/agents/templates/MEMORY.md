# MEMORY.md - Long-Term Memory

This file holds curated memory worth keeping across sessions.

## Keep

- Stable user preferences
- Durable decisions
- Long-lived project context
- Lessons that should influence future work

## Avoid

- Secrets and credentials
- Short-lived task chatter
- Raw logs that belong in `memory/YYYY-MM-DD.md`
- Duplicate facts already stored elsewhere

## Review

Periodically prune outdated entries and keep this file concise.
